# SCT_WD_2 - Stopwatch Web Application

This project is developed as part of my Web Development Internship at SkillCraft Technology.

## Project Overview
A modern and interactive Stopwatch Web Application built using HTML, CSS, and JavaScript. The application allows users to accurately measure and track time intervals with an attractive and responsive user interface.

## Features
- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Time Tracking
- Responsive Design
- Modern Glassmorphism UI
- Hover Animations

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning Outcomes
Through this project, I improved my skills in:
- JavaScript DOM Manipulation
- Event Handling
- Responsive Web Design
- UI Styling and Animations
